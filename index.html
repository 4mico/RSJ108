How to Install Adobe Fonts

Necessary Time: 3-5 minutes

This tutorial will outline the steps required in order to download and install fonts from the internet, so that you may use them in Adobe or Microsoft programs. If you have ever seen an eye-catching or interesting font online and wondered how you can use that font yourself, then this tutorial is for you!

  • Things You'll Need

  • Computer
  • Internet access
  • Adobe/ Microsoft text editing program
  • Desired font
  • A bit of time
  • Creative vision

  1. Get on the web and find a font that resonates with you and that you'd like to use.

    There are numerous resources for downloading fonts online. Do some research for websites that contain fonts that you are interested in, many websites will offer those fonts for free!

  2. Download and Unzip Font File

    Once you have found a font that speaks to you, assuming it's free, doanload it to your computer. Often when downloading a new font it will be in a zip compression format. Right click the zip file and select "unzip". Then your unzipped font file will be in a regular folder.
